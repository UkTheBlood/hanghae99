1. 해시 자료구조란? 

해시 구조란 키와 값 쌍으로 이루어진 데이터 구조
해시 구조에서는 데이터를 빠르게 찾을 수 있는 장점 (배열보다 빠름)

해시 테이블은 자료가 많아도 찾는데 걸리는 시간은 동일, 
배열은 자료가 안에 많으면 걸리는 시간이 늘어남.

같은 index에 자료를 2개 이상 넣으면 해시 충돌이 일어남

2. 자바스크립트 객체에서 주로 하는 일
여러 속성을 하나의 변수에 저장할 수 있도록 해주는 데이터 타입
 - key, value를 저장할 수 있는 구조
 - 객체가 아닌 것들 : string, number, boolean, null, undefined

3. 프로퍼티
 - 접근 가능한 이름과 활용 가능한 값을 가지는 특별한 형태
 - 특정 객체가 가지고 있는 정보를 품고 있음
 - key: 빈 문자열을 포함한 모든 문자열 또는 symbol 값
 - value: 모든 값
